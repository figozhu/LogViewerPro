# Build Resources

Place application icons here:

- **icon.ico** - Windows icon (256x256 or larger)
- **icon.icns** - macOS icon (512x512 or larger)
- **icon.png** - Linux icon (512x512)

You can use online tools to convert images:
- https://www.icoconverter.com/ (for .ico)
- https://cloudconvert.com/png-to-icns (for .icns)
